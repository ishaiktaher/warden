Warden by Vouchins

Management console

The console is served by a deployed Warden control plane because it manages agent identities, connectors, policies, credential grants, approvals, revocations, and audit evidence. The public website remains read-only and never receives production credentials.

Run the console locally

Start the full control plane, then open http://127.0.0.1:8000/console. Production deployments require OIDC and the warden:admin role.

IdentityRegister and approve agent manifests and owners.
AuthorityManage policies, grants, approvals, keys, and the kill switch.
EvidenceInspect and export the tamper-evident audit ledger.
